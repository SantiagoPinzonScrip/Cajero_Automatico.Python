# Cajero_Automatico.Python
Simulador de Cajero Automático (ATM)
Este es un proyecto hecho en Python aplicando logica diseñado para demostrar habilidades en programación funcional y manejo de archivos.

Funcionalidades:
- Persistencia de Datos: El saldo se guarda en un archivo .txt, por lo que no se pierde al cerrar el programa.
- Seguridad y Validación: Uso de bloques try/except para evitar errores si el usuario ingresa datos inválidos.
- Interfaz Limpia: Menú interactivo mediante bucles while.

Tecnologías usadas:
- Python 3.x
- Librería "os" para gestión de rutas de archivos.
